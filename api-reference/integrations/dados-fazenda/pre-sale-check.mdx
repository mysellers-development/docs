---
title: 'Verificar atendimento pre-conversao'
openapi: 'POST /integrations/dados-fazenda/pre-sale-check'
---

<Tip>
  **Caso de uso:** voce tem uma lista de clientes que ja pagaram e quer saber quais deles conversaram com a gente (IA ou atendente humano) **antes** do primeiro pagamento. Este endpoint cruza seus telefones + datas com a nossa base de mensagens e retorna, por telefone, se houve atendimento pre-conversao.
</Tip>

## Como funciona

Para cada item da lista:

1. Normalizamos o telefone (removemos `+`, espacos, parenteses, hifens)
2. Buscamos o telefone na tabela `phone_numbers` do workspace amarrado a sua API Key
3. Agregamos todas as mensagens em `unified_messages` cujo `created_at < firstPaymentDate` e `sender_type` esteja em `['ai', 'agent']`
4. Devolvemos a agregacao (primeira data, ultima data, total, canais)

Toda a consulta roda em **uma unica query SQL** usando CTE com `VALUES`, entao o tempo de resposta cresce de forma mais previsivel que fazendo N chamadas separadas.

## Autenticacao

<Note>
  Requer autenticacao via **`x-api-key`** vinculada a um workspace. O `workspaceId` e inferido automaticamente do token (voce nao precisa enviar). Requests sem API Key de workspace recebem `403 Forbidden`.
</Note>

## Timezone

<Warning>
  O campo `firstPaymentDate` e interpretado como **America/Sao_Paulo (UTC−3)** quando voce passar apenas a data (ex: `"2026-03-10"`).

  Se voce mandar `"2026-03-10"`, a gente trata como `2026-03-10 00:00:00-03:00`. Para fusos diferentes, envie em ISO 8601 com offset explicito (ex: `"2026-03-10T00:00:00Z"` para UTC).
</Warning>

## Limites

- **Max 500 telefones por request.** Acima disso retorna `400 Bad Request` — quebre em varios requests.
- **Rate limit padrao da API** (50 req / 10s). Com 500 telefones por request isso da ate 25.000 telefones processados por 10s por API Key.
- Resposta preserva a **mesma ordem** da lista de entrada.

## Exemplo

```bash
curl -X POST https://apiap.mysellers.com.br/integrations/dados-fazenda/pre-sale-check \
  -H "x-api-key: ms_live_..." \
  -H "Content-Type: application/json" \
  -d '{
    "checks": [
      { "phone": "5551999998888", "firstPaymentDate": "2026-03-10" },
      { "phone": "5548988887777", "firstPaymentDate": "2026-02-22" }
    ]
  }'
```

Resposta:

```json
{
  "results": [
    {
      "phone": "5551999998888",
      "hadPreSaleInteraction": true,
      "firstInteractionAt": "2026-01-04T14:22:10.000Z",

      "messageCount": 27,
      "channels": ["uazapi", "cloud_api"]
    },
    {
      "phone": "5548988887777",
      "hadPreSaleInteraction": false,
      "firstInteractionAt": null,
      "lastInteractionAt": null,
      "messageCount": 0,
      "channels": []
    }
  ]
}
```
