---
title: 'Chats que precisam de analise'
openapi: 'GET /chats/needs-analysis'
---

<Tip>
  Endpoint do **System Agent** (ARCH-246). Retorna chats ociosos com um numero minimo de mensagens para analise de IA. Parametros: `workspaceIds` (CSV), `idleSeconds` (default 120), `minMessages` (default 3), `limit` (default 25).
</Tip>
